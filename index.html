Campus Life


School Library

It is a place to read
The School Library is a place to read peacefully. We go there once in a week to read books. There is a wide selection of books, from pop-up children's books to Novels.


Computer Lab

A place filled of Computers
The Computer Lab has enough Computers for the entire Classroom. we go there sometimes to execute and learn new programs such as Tinkercad, Scratch, MS paint, etc.


Robotics lab

It is a place to learn Robotics
The Robotics lab is where we go to interact with robots, Make new robots, learn code and 3D Printing.
